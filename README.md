# そらリス

公開URL: <https://sora-risu.vercel.app>

空の変化に、やさしく合わせるために。

そらリスは、天気・気温・降水確率・気圧・花粉を一覧で見られることに加え、
1週間の服装・持ち物・過ごし方をやさしく提案するアプリです。

朝の「今日どうしよう」を少し減らし、低気圧の日も、雨の日も、花粉が多い日も、空とうまく付き合えるようにサポートします。

ナビゲーターは、空のすみっこに住む小さなリス「そらリス」です。

## 主な機能

- 位置情報取得・地名検索
- 天気・気温・降水確率・気圧・花粉情報の表示
- 24H / 3D / 7D / 14D の Timeline 切り替え
- 時間帯ごとのグラフ表示
- 1週間の服装・持ち物・アクション提案（アドバイスカード）
- アドバイスカードのチェック機能
- 朝・昼・夕方・夜の時間帯別カラー表示
- そらリスの毎日アドバイス（そらリスのひとこと・今日の空だより）
- 今日の季節の言葉（二十四節気・七十二候）
- PWA 対応

## 気に入っているところ

- 天気・気温・降水確率・気圧・花粉のグラフを同じ時間軸で一覧できる点
- アクションカードのストレッチ系アドバイス
- 二十四節気・七十二候が知れる豆知識感
- そらリスの画像を押すとアドバイスコメントをくれる点
- フッターの雲アニメーション
- どんぐり飛ばしのイースターエッグ

情報を見るだけでなく、そのまま生活判断につながるようにした点が気に入っています。

## 技術選定

Next.js 16（webpack） / React 19 / TypeScript / Tailwind CSS v4 / Recharts / Zustand / TanStack Query / Serwist。

Vite ではなく Next.js を選んだ経緯は `docs/spec.md` に書いています。

## 技術的な特徴

バックエンドを持たず、ブラウザ上で動作します。

位置情報を許可しなかった場合は東京を初期表示し、検索バーから場所を変更できます。

気象・地名取得には、Open-Meteo と BigDataCloud の鍵不要 API を使用しています。

PWA 対応には Serwist を使用しており、ホーム画面に追加して使えます。

## 取り扱いの注意

バックエンドは無いため、位置情報・設定・カードのチェック状態はブラウザの localStorage に保存されます。

気象・地名取得 API はブラウザから直接呼び出しています。

各気象情報から表示している提案は、あくまで生活判断のための参考情報です。医療助言ではありません。
